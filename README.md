# Loading and Sorting (data-attributes)

Таблица фильмов с автоматической сортировкой каждые 2 секунды.

## Особенности

- Сортировка по всем полям (ID, название, год, рейтинг)
- Чередование возрастания и убывания
- Хранение данных в `data-*` атрибутах
- Визуальные индикаторы направления сортировки
